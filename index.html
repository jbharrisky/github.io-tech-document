An Introduction to HTML

HTML is a standard markup language for encoding web pages.

  • What does HTML stand for? Hyper Text Markup Language.
  • It was invented by physicsit Tim Berners-Lee in 1990.
  • HTML elements are the building blocks of HTML pages.
  • Internet browers (Google, IE, Mozilla) use HTML to dispaly what you see on a webpage.
  • HTML is made up of HTML tags, which are words or abbreviations written between an opening tag (great-than sign, <) and closing tag (a forwrd slash & less-than sign, />)
  • HTML tabs label pieces of content such as "heading", "paragraph", "table", and much, much more.

Basic HTML Examples

Basic HTML Examples

  • All HTML documents start by declaring the documentation: <!DOCTYPE html>.
  • Every HTML document begins with <html> and ends with </html>.
  • The part that everyone will see is encased between <body> & </body> tags.

Code Example
<!DOCTYPE>
<html>
<body>
<h1> My First Heading </h2>
<p> My First Paragraph. </p>
</body>
</html>

HTML Headings

  • HTML headings have different sizes, they range from <h1> to <h6>.
  • The higher the heading number the more important the content.

Code Example
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

HTML Paragraphs

  • Paragraphs in HTML are defined using the <p>tags.

Code Example
<p>This is a paragraph</p>
<p>Yet another paragraph</p>

HTML Links

  • Links in HTML are defined using the <a> tags.

Code Example
<a> href ="https://www.google.com">Link to Google</a>

HTML Buttons

  • Buttons in HTML are defined using the <button> tags.

Code Example
<button>Click Me</button>

HTML Images

  • Images in HTML are defined using the <img> tags.
  • A Source file <src>, alternative text <alt> will need to be provided as well.

Code Example
<img src ="test.jpg"
alt ="test.com">
